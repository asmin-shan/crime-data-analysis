Crime Insights Dashboard

An interactive Power BI dashboard analyzing crime data to uncover patterns and trends in criminal activities.
It visualizes crime distribution across categories, locations, and time periods, providing insights into crime patterns and helping users better understand public safety data.

Dashboard Highlights

Crime distribution by category

Location-wise crime analysis

Year-wise crime trends

Crime comparison across different categories

Interactive filters for crime type, year, and location

Insights

Certain crime categories occur more frequently than others.

Crime rates vary depending on location.

Some years show a higher number of reported crimes.

Patterns in crime types can be observed across different regions.

Tools Used

Power BI – Dashboard design and data visualization

Excel / CSV – Data storage and preprocessing

Git – Version control

GitHub – Project hosting using GitHub

How to Use

Download the file

Open it using Power BI Desktop.

Use filters and visuals to explore the crime data interactively.

Dataset Source

Public crime dataset used for educational data analysis.
